# Python ka Chilla 2026

Welcome to the **Python ka Chilla 2026** repository! This folder contains all the learning materials, scripts, and Jupyter notebooks from the Python course.

## 📂 Repository Structure

- **`00_resources/`**: Contains helpful resources like Markdown cheat sheets.
- **`01_basics_of_python/`**: Basic Python scripts and fundamentals to get started.
- **`02_variables/`**: Jupyter Notebooks covering core concepts such as:
  - Variables & Data Types
  - Type Casting
  - Sequences & Indexing
  - Control Flow Statements
  - Functions & Lambda Functions
  - User Inputs
- **`03_python_intermediate/`**: Intermediate Python concepts including print methods, generators, and working with output files.

## 🚀 Getting Started

To explore the contents:
1. Clone or download this repository.
2. Ensure you have Python and Jupyter Notebook installed.
3. Open the `.ipynb` files in your preferred environment (like VS Code or JupyterLab) and run the cells to see the output.
